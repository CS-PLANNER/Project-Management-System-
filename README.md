# Project-Management-System-
-Addon FSD final project.
 Overview
A modern web-based Project Management System designed to streamline team collaboration, task tracking, and project organization.

 Features
Dashboard - Real-time project overview
Sprint Management - Agile sprint planning and tracking
Task Management - Create, assign, and track tasks
User Authentication - Secure login and role-based access
Real-time Updates - Live collaboration features

Tech Stack
Frontend: React.js, CSS3, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT Tokens

Future Enhancements
1. Automated Employee Onboarding
  Send automatic welcome email with credentials when admin creates employee profile
  Include username/password in email for new employees
  Email validation system to verify employee email addresses
2. Enhanced Admin Dashboard
  User Management Panel: View all users with filtering by name and role
  Quick View Feature: View button to see detailed user information
  Role-based User Lists: Filter users by specific roles (Developer, QA, PM, etc.)
  Bulk Operations: Select multiple users for batch actions
3. Advanced Search & Filtering
  Sprint Search: Filter sprints by start/end date ranges
  Task Filtering: Search tasks based on creation/due dates
  Multi-criteria Search: Combine filters (date + assignee + status)
  Export Filters: Export filtered results as CSV/PDF
A. Analytics & Reporting
  Project progress dashboards with charts/graphs
  Team performance metrics
  Sprint velocity tracking
  Time tracking integration
B. Communication Features
  In-app messaging/notifications
  Comment threads on tasks

C. Project Management Tools
  Gantt chart view for project timelines

  Risk assessment matrix

D. Security & Compliance
  Two-factor authentication (2FA)

  Dark/Light theme toggle
